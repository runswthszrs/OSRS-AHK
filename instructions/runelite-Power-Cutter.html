Pre-Requirements:

  • Runelite
  • Woodcutting level per tree
  • Shift-Drop MUST be enabled
  • Brightness on second notch in game ( ? ? ? ? ) <? where the dash is
  • Exp drops set to right side, white color, largest text size, permanent duration ? (XP Button next to HP, not RL plugin)

Required Plugins:

  • Camera (Off)
  • Woodcutting (On)
  • Mouse Tooltips (On)
  • Entity Hider (On) ? Hides other players
  • Object Marker (On) ? with the settings below
  • Inventory Tags (On) ? with the settings below

Setup:
  • Logs in your inventory must be tagged with inventory group 4
  • All tress you want to cut must be marked with the ?Object Marker? (Shift + Right Click)
  • The ?Woodcutting? plugin overlay must appear in the top left, any plugins obstructing this must be disabled (Example: Slayer/Timers Plugin)

Note: The first and second inventory slot DO NOT GET DROPPED. This is designed so you can put your axe / teleports in the inventory.
