Setup Requirements:

  • Demon Butler
  • Hammer & Saw in inventory
  • Oak/Mahogany noted planks in inventory
  • ?Servant?s Money? bag must be built in your house, with money in it

Setup:

  • Table must be un-built upon script start
  • Un-built Oak/Mahogany Table must be marked with ?Object Marker?
  • Built Oak/Mahogany Table must be marked with ?Object Marker?
  • Demon Butler must be tagged with ?NPC Indicators?
  • Request 24 Oak/Mahogany Planks while standing next to the table you wish to build
  • Start script once he returns

Note: Make sure you have the table horizontal (East/West) and not vertically, otherwise the script will not work consistently
