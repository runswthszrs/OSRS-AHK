You must do at least do the minigame once before botting it (takes about 1minute, buy some arrows and shoot the target until reaching 650 points, then start the script.

Requirements:
40 Range
Gold Coins (200gp per 10 arrows)
First Inventory Spot Empty for arrows.
?Tag? the ?Competition Judge? NPC with the NPC Indicator plugin (Shift+Right Click to tag) ? Refer to plugin settings below
?Tag? the archery ?Target? with the Object Marker plugin (Shift+Right Click to tag) ? Refer to plugin settings below
?Esc to close current inferface? enabled in the in-game hotkeys settings

Required Plugins:
Object Markers (On) ? with the color of the object marker set to #FFFF00
Npc Indicators (On) ? with the highlight color set to #00FFFF
Camera (Off)
Interface Styles (Off)
