Pre-Requirements:

  • Runelite
  • Any raw food and the respective cooking level required
  • Brightness on second notch in game ( ? ? ? ? ) <? where the dash is
  • Escape must be set to close current interface ? (Set in hotkeys, as shown below)
  • Bank placeholders enabled ? Required for the script to log you out once you run out of raw food
  • Exp drops set to right side, largest text, faster speed, 3 secs duration, most recent counter/Progress, white color ? (XP Button next to HP)

Required Plugins:

  • Camera (Off)
  • Mouse Tooltips (On)
  • Inventory Tags (On)
  • Menu Entry Swapper (On)
  • Entity Hider (On) ? with settings shown below
  • Object Marker: (On) ? with the ?highlight color? set to #FFFF00
  • NPC Indicators (On) ? with with the ?highlight color? set to #00FFFF and ?Highlight Menu Names? off

Setup:

  • Cook quantity must be set to ?All?
  • Bank ?Quantity? option must be set to ?All?
  • Designated food must be in first bank slot ? As shown below
  • The food you are cooking marked with group 1 inventory tag ? As shown below
  • Bank NPC which you wish to use needs to be tagged? (Shift + Right click on the npc)
  • Campfire must be marked as shown in the picture above ? (Shift+Right Click to mark)
  • Bank NPC which you wish to use needs to be tagged? (Shift + Right click on the npc)
  • Must be standing in the location shown before starting the script. (Rogues Den ? Beneath Burthorpe?s Pub ? 1 step east of the banker)
