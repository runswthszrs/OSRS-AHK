Purpose of the script:
Up to 12k strength exp per hour without gaining hitpoints experience, ideal for making pures for PvP!

How to use: 
Go to world 319 (Blast Furnace pumping world, people will maintain the pump)
Set an object marker on the pump
Ensure correct exp-drop settings. (Refer to above picture)
Make sure exp drops are enabled (By clicking on the ?XP? button next to the minimap)
Start the script when standing next to the pump.

Required Plugins

Interface Styles (Off)

Mouse Tooltips: (On)

Camera Zoom: (Off)

Object Marker: (On) ? with the following highlight color (Yellow FFFFOO)
