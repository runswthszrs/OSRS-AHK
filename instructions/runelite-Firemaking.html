Pre-Requirements:
  • Runelite
  • Brightness on second notch in game ( ? ? ? ? ) <? where the dash is
  • Escape must be set to close current interface ? (Set in hotkeys, as show below)
  • Exp drops set to right side, white color, largest text size, 3 second duration ? (XP Button next to HP, not RL plugin)

Plugin Requirements:

  • Camera (Off)
  • Inventory Tags (On)
  • Mouse Tooltips (On)
  • Menu Entry Swapper (On)
  • Entity Hider (On) ? Hides other players
  • Ground Markers (On) ? With ?Remember Color per tile? enabled in the settings
  • NPC Indicators (On) ? with with the ?highlight color? set to #00FFFF and ?Highlight Menu Names? off
  • GPU (Off)
  • Minimap (On) ? With the ?Item? color as ANYTHING EXCEPT red (Example being #FF7D00 for orange)

Setup:

  • Logs in first bank spot
  • Tinderbox in first inventory spot
  • Start at Falador east bank as shown in the picture above
  • Bank left click quantity withdraw option must be set to ?All?
  • Logs of choice marked with group 1 inventory tag ? As shown below
  • 3 Ground Markers must be placed ? As shown below

Note:

Red Ground Marker Color: FF0000 (Some find marking the bank booth instead of the tile works better for them)
Purple Ground Marker Color: FF00FF (PURPLE TILE MARKER MUST BE PLACED BEFORE THE GREEN ONE)
Green Ground Marker Color: 00FF00 (PURPLE TILE MARKER MUST BE PLACED BEFORE THE GREEN ONE)

Potential bugs: This script is highly reliant on a stable internet connection and relatively low latency, if you have a temperamental connection, this script will not work reliably.
