Requirements:

Access to the Warriors guild.
Energy Potions
Food

Required Plugins:

Mouse Tooltips (On)
Inventory Tags (On)
Object Markers (On) ? With highlight color set to FFFF00
Interface Styles (Off)

How to Use:
Mark one of the kegs with an object marker.
Mark all doses of energy potions with inventory tag 4 (Purple)
Mark food with inventory tag 1 (Red)
Stand next to keg
Start Script.

??[OPTIONAL]??
If you want the script to logout if a player is detection, enable the ?Minimap? plugin and change the ?Player? color to #00FFFF

Notes: This script does not bank, and will log out in the scenario of running out of energy potions or food.
