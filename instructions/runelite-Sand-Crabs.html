Requirements:

Runelite Client*

Runescapes Exp Counter must be showing (XP drop must be white text)

4 Tile markers need to be set up (Refer to the video at the bottom as to where to place them, aslong as next tile marker is visible on the minimap) ? To set a ground marker, hold shift+right click.

(Refer to this youtube video at the bottom of the page for help with ?Ground Markers?)

Ground Marker Colour Codes

Cyan = 00FFFF
Blue = 0000FF
Green = 00FF00
Purple = FF00FF
(Coloured tiles need to be placed in that order)

Optionally, you can use potions and food, they will need to be tagged in your inventory with the ?Inventory Tags? Plugin
Potions = with group 4 (Purple)
Food = Group 2 (Green)
