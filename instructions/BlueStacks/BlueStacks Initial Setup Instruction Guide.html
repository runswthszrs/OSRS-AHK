System setup instructions:

  • Correct version of BlueStacks: BlueStacks 4.110.0.1081 – NOT THE LATEST VERSION OF BLUESTACKS – After installing and opening this version of bluestacks for the first time, it will prompt you to update, just close the update window.
  • Application scale size (Right click desktop -> Display Settings) must be set to 100%. ( YOU must reboot windows to take effect)

Bluestacks client setup instructions:

  • Open Bluestacks settings via the gear icon up top
  • Click on “Display” -> Set resoulution to 1280×720, and 240 DPI
  • Click on “Engine” -> Set to “OpenGL”, Advanced graphics engine off, 2-4 CPU cores, 1024-2048MB Ram.
  • Click on “Notifications” -> Uncheck everything.
  • Click on “Preferences” ->  Uncheck everything, then enable Always on top and close webpages when running games.

Bluestacks in-game instructions:

  • Download the Oldschool Runescape app (Not Runescape as this is RS3)
  • Open the bots and click anywhere in game in bluestacks
  • Press Shift+F1 to automatically resize BlueStacks to the proper size
  • Press Shift+F2 to angle your camera north and click your compass (If it misses the compass something is wrong)
  • In game brightness must be on the second notch for all scripts:

 

  • Click the RS settings wrench -> Controller -> User Interface slider must be fully to the left:
