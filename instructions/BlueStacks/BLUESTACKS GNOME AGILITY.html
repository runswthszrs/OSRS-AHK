Gnome Course Agility:
Requirements:
Experience drop duration must be changed to 1 second.
Exp Drop ‘Counter’ must be set to ‘Total XP’
Exp Drop ‘Speed’ must be set to ‘Fastest’
Fully Zoomed Out
Exp Drops Enabled
Must be standing in the location above before starting the script.

How to use:
(As this is one of my more complicated scripts, there will be a video tutorial at the bottom of the page alternatively)
Starting at the end of the agility course (Just after the last obstacle) You must hover your cursor over the obstacles and set the coordinates, upon setting the coordinates, you’ll hear a confirmation beep. You can set/save these coordinates again if they become inaccurate.

First Obstacle = Ctrl+Alt+1
Second Obstacle = Ctrl+Alt+2
Third Obstacle = Ctrl+Alt+3
Fourth Obstacle = Ctrl+Alt+4
Fifth Obstacle = Ctrl+Alt+5
Sixth Obstacle = Ctrl+Alt+6
Seventh Obstacle = Ctrl+Alt+7
