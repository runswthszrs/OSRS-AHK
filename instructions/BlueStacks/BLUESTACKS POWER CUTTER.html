Requirements:
BlueStacks
Any location where you have access to 2 willow/teak trees without moving.
Fully-Zoomed in (In settings above brightness)
Required Woodcutting level for
Click-to-drop ENABLED
Last 27 Inventory spaces free (For Willow/Teak Logs)
How to use:
Press CTRL+ALT 1 and 2 in the center of each willow/teak tree to designate where they are located. (Fully Zoom in First!)

First Willow/Teak Tree = Ctrl+Alt+1
Second Willow/Teak Tree = Ctrl+Alt+2
Then Start the script!

Please note that this script has not been extensively tested with the other type/shape of willow tree, so it’s advised to use it in the above location (South-west of Catherby Bank) if you can’t get it working!
