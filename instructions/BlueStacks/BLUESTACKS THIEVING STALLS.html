Note: This script is mainly designed for Fruit/Tea stalls (Stalls where you won’t get attacked by anything) as the script won’t eat/protect you from any aggression.

Requirements:
Top-left/First inventory spot must be empty.

‘Tap to Drop’ must be enabled in the in-game settings (As shown in the picture above).

You must set the coordinates of the thieving stall by hovering your cursor over the stall and pressing Shift+F3 (You’ll hear a beep to confirm it) – The BlueStacks window MUST be active before doing this. – If this coordinate ever becomes inaccurate, you can re-set it again using Shift+F3 to asign the coordinate again.

Note: You want to press F3 in a location on the stall where if the bot clicks the stall when there’s nothing to be stolen, it won’t run anywhere.
